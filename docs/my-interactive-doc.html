Phase-Locked Production
No phase begins without an approved upstream artifact. Textbook must be finalized before MCQs. MCQs must be approved before the slide outline. The agent enforces this gate at every step.
🤝
One Decision Gate Per Message
Claude asks one question at a time and waits. No bulk decisions, no assumed approvals. Every connector action — Monday update, Gamma creation, file save — requires explicit user confirmation.
📐
Depth Calibration First
Before generating any content, Claude runs depth calibration — analyzing scope, cognitive level, and exam weight. Output length and structure are determined by the calibration result, not guessed.
🗄
Monday-First Storage
Textbooks → Word Doc → Monday Textbook File Column. MCQs → Excel → Monday MCQ File Column. Presentations → Gamma URL + PPTX → Monday. Scripts → Doc → Monday Slide Script Column.
✂️
Revise Only What's Asked
Revisions target only the flagged section, question, or cue. All approved content outside the revision area is preserved exactly. The agent never rewrites unflagged sections.
⚡
Token Optimization Built In
Claude batches content generation into chunks, pauses for mid-gen checks, and references prior sections rather than repeating them. Comprehensive depth always triggers chunking.
