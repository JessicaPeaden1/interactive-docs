Demo Overview

This demo shows how a user works with the HOCK AI Agent to create training materials.

The AI guides the process through structured phases:

  • Textbook Creation
  • MCQ Creation
  • Slides / Presentation
  • Script Creation
  • Video Production

The trainer should demonstrate each phase using a real example.

Phase 1 – Textbook Creation

Trainer Prompt Example:

Create a textbook for CIA Part 2 – Study Unit 5: Risk Factor Identification and Evaluation.

The AI generates a textbook draft.

Trainer demonstrates revision request:

Revise section 3 and add a real-world example.

Once satisfied:

Approve the textbook.
Phase 2 – MCQ Creation

The AI automatically creates multiple choice questions.

Trainer demo prompt:

Make the questions slightly harder and add explanations.

Optional request:

Convert the MCQs to Excel.

Final approval:

Approve MCQs.
Phase 3 – Slide / Presentation Creation

The AI generates a structured slide deck outline.

Trainer prompt:

Reduce text on the slides and add one summary slide.

When approved the presentation is created.

Approve slides.
Phase 4 – Script Creation

The AI generates instructor narration for each slide.

Trainer prompt:

Make the script more conversational.

Optional request:

Add teaching examples for difficult topics.

Final approval:

Approve script.
Phase 5 – Video Creation

The script and slides move into production.

Trainer explains that video creation may occur in tools such as:

  • HeyGen
  • Internal video production tools

The agent then updates the project to complete.

Trainer Tips
  • Always show one real revision during demo
  • Explain that approvals control progress
  • Keep the demo under 10 minutes
  • Highlight that users can ask natural questions
