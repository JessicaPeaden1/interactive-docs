Phase 1 · Automation Detail
How it works
Three automation workflows run in sequence — each building on the output of the previous. Together they handle everything from email arrival to Tom's morning review queue.
1
Email Triage & Data Extraction
Days 1–5
TRIGGER: New email arrives in Outlook → AI: Is this an ITB? (classified in seconds) → If yes: AI: Extract project name, GC, bid due date, contact, drawing links, format type → Record: Structured bid entry added to register
✓ Every ITB email captured as a structured record — no manual reading or data entry
2
Drawing Retrieval
Days 4–12
PDF attachments: AI identifies HVAC drawing pages and summarizes scope → Dropbox packages: AI matches contractor job names to Dropbox folder names, navigates to HVAC files, generates direct links → Portal links: Credentials and project IDs extracted — flagged for manual portal access
✓ Drawing links surfaced before Tom opens the review — no searching, no Dropbox navigation
3
Morning Review Digest
Days 10–15
TRIGGER: 6:45am daily → Read all pending bids, sorted by urgency → AI: Assemble morning email — project cards with drawing links, scope summary, due date → Tom clicks BID / PASS / FLAG → Decision recorded, bid calendar updated automatically
✓ Tom's entire bid decision workflow in one 15-minute email review — all data already organized
Phase 1 · Design Principle
What stays human
Automation handles the searching, sorting, and organizing. Every decision that requires judgment, experience, or a customer relationship stays with the Hilberts team. This is the design — not replacing people, relieving them.
✋
Bid / Pass Decision
AI presents every bid with drawings located and scope summarized. Tom reviews and clicks BID, PASS, or FLAG. The decision is always his — automation eliminates the search work, not the judgment.
👁
Proposal Review
In Phase 2, every AI-generated or vendor-generated proposal is reviewed internally before it reaches a customer. No proposal leaves without a human sign-off.
📞
High-Value Relationships
Large opportunities and key accounts trigger personal call alerts rather than automated emails. AI manages routine follow-up — people manage important relationships.
🚩
Flagged & Uncertain Items
Anything the system cannot handle with high confidence — portal-only bids, unmatched Dropbox folders, low-confidence extractions — is flagged for manual review rather than processed automatically.
📊
Strategic Decisions
Which markets to pursue, which customers to prioritize, which products to lead with — all of this remains with Tom and the team. AI provides data to inform those decisions, not make them.
🔍
Quality Control
HVAC scope summaries, room schedule data, and extracted bid fields can all be reviewed and corrected by the team before acting on them. Automation is a first pass — not a final word.
